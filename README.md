# 🎨 Portfolio Website

React와 TypeScript를 사용하여 구현 중

## 📂 Project Structure (현재 구조)

기능별로 폴더를 분리해서 관리하는 중

src/
├── components/
│ ├── Header/ # 상단 메뉴바 (로고, 네비게이션, 언어 변경 버튼)
│ │ ├── Header.tsx
│ │ ├── Header.css
│ │ └── logo.png
│ ├── Home/ # 메인 히어로 섹션 (인사말, 소개)
│ │ ├── Home.tsx
│ │ └── Home.css
│ └── About/ # 자기소개 섹션 (프로필 사진, 인적 사항)
│ ├── About.tsx
│ └── About.css
├── App.tsx # 메인 페이지 조립
└── App.css # 전체 공통 스타일
